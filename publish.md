# How to publish changes to @spike/api, @spike/global, @spike/pdf-cli

```sh
cd /spike/v8/pub
yarn run publish # NOTE: NOT `yarn publish`
```

## How to update @spike/priv to use latest public libs

```sh
cd /spike/v8/priv
# yarn add -W @spike/api @spike/global
yarn install
```
